Perguntas e respostas

  1. Na tag <input> com atributo type="text", para que serve o atributo" placeholder"?
    Resposta: serve para adicionar um texto temporário, depois que o usuário começa a escrever ele desaparece.

  2. Na tag <input> com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?
    Resposta: serve para que o usuário da aplicação selecione uma ou mais alternativas entre as opções disponíveis.

  3. Na tag <input> para que serve o atributo "value"?
    Resposta: serve para definir um valor inicial para o input, caso ele seja do tipo text.

  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?
    Resposta: o atributo name.

  5. Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?
    Resposta: serve definir um rótulo para muitos elementos do formulário e ajuda usuários que têm dificuldade em clicar em regiões muito pequenas ex: botão e caixas de seleção.

  6. Na tag <option> para que serve o atributo "selected"?
    Resposta: serve para criar uma lista suspensa é usado em um formulário para coletar a entrada do usuário.

  7. Nas tags de formulário para que serve o atributo "required"?
    Resposta: serve para indicar que é um campo obrigatório.

  8. Para que serve a tag <textarea>?
    Resposta: serve para criar um caixa de texto de várias linhas.

  9. Na tag <form> para que serve o atributo "action"?
    Resposta: serve para especificar para onde os dados do formulário devem ser enviados.

  10. Na tag <form> para que serve o atributo "method"?
    Resposta: serve para especificar o método http a ser usado ao enviar os dados do formulário, get ou post.

  11. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form> ?
    Resposta: post

  12. Qual valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos?
    Resposta: post

  13. O que as tags <fieldset> e <legend> produzem?
    Resposta: fieldset serve para agrupar dados relacionados em um formulário e legend serve para definir uma legenda para o fieldset elemento.
